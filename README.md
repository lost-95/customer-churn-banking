# Customer Churn Analysis (Banking)

This project analyzes customer churn in a banking context, focusing on understanding the key drivers of customer attrition through exploratory data analysis and a baseline classification model.

The objective is to extract clear, interpretable, and business-oriented insights that simulate a real-world analytics workflow.

## Project Structure
- `notebooks/` – exploratory data analysis and modeling steps  
- `data/` – raw data (not versioned)  
- `reports/figures/` – exported visualizations  

## Dataset
The dataset is publicly available on Kaggle:
https://www.kaggle.com/datasets/mathchi/churn-for-bank-customers

To run the analysis, download the dataset and place `churn.csv` inside the `data/` folder.

## Tools
- Python
- pandas, NumPy
- matplotlib, seaborn
- scikit-learn
- Jupyter Notebook
